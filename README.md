# abimorrownet
abimorrownet is a social network developed for the students of [abimorrowland](https://abimorrowland.de/).

## Features
* Profiles
    * Uploading images
    * Posting private comments
    * Adding attributes
* Committees
    * Associating roles
    * Creating votings
* Teachers (cp. 'Profiles')
    * Adding private comments
    * Adding attributes
